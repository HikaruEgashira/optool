# optool

A command-line tool for calc

```
optool 0.1.0
A command-line tool for string substitutions

USAGE:
    optool <op> <a> <b>

OPTIONS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <op>    operator
    <a>     number
    <b>     number
```
