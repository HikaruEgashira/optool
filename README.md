# optool

A command-line tool for calc

```
optool 0.1.0
A command-line tool for string substitutions

USAGE:
    optool <op> <a> <b>

OPTIONS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    <op>    operator
    <a>     number
    <b>     number
```

## usage

```
optool Add 3 4
7

optool Sub 3 7
-4

optool Mul 5 6
30

optool Div 17 3
5
```
